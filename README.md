# 📄 Resume Builder

A modern, feature-rich **Resume Builder** web application built with **React + Vite** and **Tailwind CSS**. Create stunning, professional resumes in minutes with real-time preview, multiple templates, drag-and-drop reordering, and one-click PDF download.

---

## ✨ Features

- 🎨 **10 Professional Templates** — Modern Dark, Classic Pro, Minimalist, Bold Designer, Elegant, Executive, Corporate Blue, Creative, Compact, and Tech Dev
- 👁️ **Live Preview** — See your resume update in real time as you type
- 🖨️ **PDF Export** — Download your resume as a print-ready PDF with one click
- 🌗 **Dark / Light Mode** — Toggle between dark and light themes
- 🗂️ **Drag & Drop Reordering** — Reorder resume sections with `@dnd-kit`
- 💾 **Auto-Save** — Resume data persists in `localStorage` automatically
- 🚀 **Demo Data** — Load sample resume data instantly to preview templates
- 📱 **Mobile Responsive** — Fully responsive with a tab-based Edit / Preview mode on mobile
- ✍️ **Rich Sections** — Personal Info, Summary, Experience, Education, Projects, Skills, and Certifications

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/vinaysunkari30/Resume-Builder.git
```

> Or if you already have the project folder, just navigate into it:
> ```bash
> cd path/to/ResumeBuilder
> ```

### 2. Install Dependencies

```bash
npm install
```

This will install all packages listed in `package.json`, including React, Vite, Tailwind CSS, and all other dependencies.

### 3. Start the Development Server

```bash
npm run dev
```

The app will start and be available at:

```
http://localhost:5173
```

## 🧩 How to Use

1. **Open the app** at `http://localhost:5173`
2. **Load Demo Data** — Click "Load Demo" in the navbar to populate sample data and preview templates quickly
3. **Fill in your details** — Use the left panel to enter your Personal Info, Summary, Experience, Education, Projects, Skills, and Certifications
4. **Switch Templates** — Click any template in the top template switcher bar to change your resume's look
5. **Toggle Dark / Light Mode** — Use the moon/sun icon in the navbar
6. **Download PDF** — Click the "Download PDF" button in the navbar to save your resume
7. **Reset** — Click "Reset" twice to clear all data and start fresh
